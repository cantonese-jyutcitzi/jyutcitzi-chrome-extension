# Jyutcitzi Chrome Extension

1. Download `jyutcitzi-chrome-extension-dist.zip` from [Releases](https://github.com/cantonese-jyutcitzi/jyutcitzi-chrome-extension/releases).
2. Unzip it. You should get **one folder** (usually named `**jyutcitzi-chrome-extension-dist`**) whose **root** (what you see when you open that folder) already contains `**manifest.json`**, plus `content.js`, `yaml/`, `fonts/`, and the rest — not a parent folder that only contains this folder as a nested item. If unzipping put those files loose in `Downloads` (no folder), create a new folder, move everything into it, and treat that folder as `**jyutcitzi-chrome-extension-dist**` in the steps below.
3. Open `chrome://extensions`.
4. Turn on Developer Mode.
5. Click “Load unpacked”.
6. In the file picker, **select that folder itself** — the `**jyutcitzi-chrome-extension-dist`** extension root — so Chrome loads the directory where `**manifest.json` is a direct file inside it**.
